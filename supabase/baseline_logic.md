# Remoralization baseline logic

## Pillars
- Body
- Order
- Responsibility
- Mastery

## Baseline structure
- 4 pillars
- 4 scored questions per pillar
- 16 scored questions total
- each scored question uses a 1 to 5 scale

## Score calculation
- add the 4 question scores inside each pillar
- convert each pillar to a score out of 25
- total score = Body + Order + Responsibility + Mastery
- total score is out of 100

## Result generation
The baseline must generate:
- total_score
- body_score
- order_score
- responsibility_score
- mastery_score
- strongest_pillar
- weakest_pillar
- diagnosis_type
- diagnosis_summary
- main_bottleneck
- recommended_challenge
- next_move
- share_text

## Challenge rule
The weakest pillar determines the first 14-day challenge.
