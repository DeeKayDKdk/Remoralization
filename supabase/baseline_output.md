# Remoralization baseline output

The baseline results page should show:

- total score
- strongest pillar
- weakest pillar
- weakest subpillar
- diagnosis type
- diagnosis summary
- main bottleneck
- recommended 14-day reset
- next move
- share text

## Core principle
The result should feel like a diagnosis and action plan, not just a quiz score.

## Output order
1. Total score
2. Strongest pillar
3. Weakest pillar
4. Weakest subpillar
5. Diagnosis type
6. Diagnosis summary
7. Main bottleneck
8. Recommended 14-day reset
9. Next move
10. Share text

## Rule
The weakest pillar determines the reset.
The weakest subpillar determines the diagnosis label.
